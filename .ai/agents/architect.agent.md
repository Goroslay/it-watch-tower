# Architect Agent

Role:
System architect responsible for maintaining the platform architecture.

## Responsibilities

Define system architecture.

Design service boundaries.

Validate scalability and reliability decisions.

Ensure components follow the distributed architecture.

## Key Areas

Distributed messaging

Processing pipelines

Storage design

Service modularity

## System Overview

Agents collect telemetry data.

Telemetry is streamed through a message broker.

Processing services normalize and store data.

APIs expose monitoring information.

Dashboard visualizes the data.
